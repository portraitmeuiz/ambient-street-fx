거리 소음 → 앰비언트 FX

마이크로 들어오는 거리의 소음을 실시간으로 필터·잔향·딜레이·코러스 체인에 통과시켜 부드러운 앰비언트 사운드로 바꿔줍니다. 반드시 이어폰을 착용하세요 (스피커 사용 시 피드백 하울링 발생).

마이크 대기 중
톤 / 게이트
공간감 (리버브)
에코 / 코러스
작곡 레이어 (앰비언트 코드 패드)

Cmaj9 → Em7(add11) → Am9 → Fmaj7(#11) 진행을 천천히 순환하는 신스 패드입니다. 낮은 벨로시티 · 롱 어택/릴리즈로 크로스페이드되어 해결감보다 부유하는 느낌을 냅니다.

Web Audio API 기반 PWA · 홈 화면에 설치해 앱처럼 사용 가능 · 브라우저 마이크 권한만 허용하면 바로 동작합니다.
